隐私协议

最后更新日期：2023年10月24日

1. 信息收集

在您使用 Huozi Writer 的过程中，我们可能会收集您的基本账户信息、创作内容元数据以及设备访问记录。这些信息旨在为您提供更优质的排版与编辑体验。我们郑重承诺，绝不主动泄露您的文学创作内容。

个人身份信息

姓名、电子邮件地址、支付凭证（仅用于订阅服务）。

使用数据

登录频率、功能偏好、应用崩溃日志及性能统计。

2. 数据安全与存储

您的文字是思想的结晶。我们采用端到端加密技术保护您的草稿。所有存储在云端的数据均分布于具备军工级安全保护的数据中心，并定期进行冗余备份，以确保在任何极端情况下您的心血不被抹除。

“我们视数据安全为文人的笔墨纸砚，不容染尘，不容遗失。”

3. 第三方服务

为了提供高级排版导出（如 PDF 生成）和云端同步，我们可能与受信任的第三方服务商合作。我们仅在必要范围内与其共享加密后的匿名数据，且这些服务商均受严格的保密协议约束。

no_accounts

匿名处理

除非法律要求，否则我们不对外提供可识别身份的数据。

delete_forever

彻底删除

一旦注销账户，所有个人数据将在30个工作日内永久抹除。

security

自主控制

您可以随时在设置中关闭数据分析或同步功能。

4. 联系我们
