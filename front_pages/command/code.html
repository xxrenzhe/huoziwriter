《长河落日》

塞外的风沙总是带着一种宿命的沉重。李长歌站在城墙上，望着远方地平线上那一抹残阳。手中的剑柄微微发烫，仿佛在回应着远方的马蹄声。

“将军，突厥的先遣队已经到了十里外。”副将沉声禀报，盔甲的摩擦声在寂静的黄昏显得格外刺耳。
